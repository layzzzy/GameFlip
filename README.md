## 🚀 部署到 GitHub Pages

### 方法一：自动部署 (推荐)

1. Fork 这个仓库到您的 GitHub 账户
2. 进入仓库设置 (Settings)
3. 找到 "Pages" 选项
4. 在 "Source" 中选择 "GitHub Actions"
5. 推送代码到 main 分支即可自动部署

### 方法二：手动部署

1. 克隆仓库到本地：
```bash
git clone https://github.com/yourusername/GameFlip_Website.git
cd GameFlip_Website
```

2. 进入 GitHub 仓库设置，启用 GitHub Pages
3. 选择 main 分支作为源
